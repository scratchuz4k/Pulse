## Summary

Created Server and ServerMember DB entities, updated Room to reference a Server via ServerId FK, removed the old `CreatedByUserId` stop-gap, registered the new DbSets in AppDbContext, created ServersController with all five server-scoped endpoints, and pruned the global room endpoints from RoomsController.

- `Pulse.Server/Models/Room.cs` — Added `ServerId` (int, non-nullable FK); removed `CreatedByUserId` stop-gap
- `Pulse.Server/Data/AppDbContext.cs` — Added Servers/ServerMembers DbSets; composite unique index on Room(ServerId+Name); Server FK with cascade delete; ServerMember unique index on (ServerId,UserId)
- `Pulse.Server/Controllers/ServersController.cs` — POST /servers, POST /servers/join, GET /servers/me, GET /servers/{id}/rooms, POST /servers/{id}/rooms (owner-only 403)
- `Pulse.Server/Controllers/RoomsController.cs` — Removed GetRooms and CreateRoom; kept GetRoomToken
