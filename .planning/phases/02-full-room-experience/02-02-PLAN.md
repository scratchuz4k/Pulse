## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Renderer → PresenceHub MuteChanged | Authenticated SignalR connection; client invokes with bool |
| PresenceHub → Group broadcast | Server fans out to all clients in room group |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02B-01 | Spoofing | MuteChanged — who is muted | mitigate | Hub uses Context.ConnectionId (server-assigned) as the key; the bool isMuted is the payload but the identity comes from the server side |
| T-02B-02 | Tampering | Client sends MuteChanged for another user | accept | SignalR [Authorize] + ConnectionId as source of truth; client cannot forge another user's connectionId |
</threat_model>
