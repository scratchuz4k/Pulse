  <action>
In PresenceHub.cs:
- Add a public static helper `GetRoomParticipants(string roomName)` returning a lightweight shape per participant `{ displayName, userId }` (or empty list if room absent). Use it to build list payloads. Add a static `BuildRoomListPayload(IEnumerable<(int Id, string Name)> rooms)` that maps each room name to its participants via `_rooms`, producing `{ id, name, participants: [{ displayName, userId }] }`.
- Inject `AppDbContext` into the hub via constructor so LeaveRoom/OnDisconnected can delete empty rooms. Change class to `public class PresenceHub(AppDbContext db, IHubContext<PresenceHub> hubContext) : Hub` (inject IHubContext to broadcast RoomListUpdated from the hub after deletion â€” Clients.All is available on the hub directly via `Clients.All`, so IHubContext injection is optional; prefer using the hub's own `Clients.All`).
- In LeaveRoom, after removing the participant: if `_rooms.TryGetValue(roomName, out var room)` and `room.IsEmpty`, then `_rooms.TryRemove(roomName, out _)`, delete the DB row (`var dbRoom = await db.Rooms.FirstOrDefaultAsync(r => r.Name == roomName); if (dbRoom != null) { db.Rooms.Remove(dbRoom); await db.SaveChangesAsync(); }`), and broadcast an updated room list: query remaining `db.Rooms` ordered by name, build the enriched payload via `BuildRoomListPayload`, and `await Clients.All.SendAsync("RoomListUpdated", payload)`. This delivers the D-09 auto-delete-on-empty behavior.
- Apply the same empty-room cleanup in OnDisconnectedAsync for the room the connection was in (it iterates `_rooms`; after `room.TryRemove(...)` succeeds, check `room.IsEmpty` and run the same delete+broadcast).
- Extract the delete-if-empty + broadcast into a private async helper to avoid duplication.

In RoomsController.cs:
- Change `GET /rooms` to build the enriched payload: fetch `db.Rooms.OrderBy(r => r.Name).Select(r => new { r.Id, r.Name })`, then map each into `{ id, name, participants = PresenceHub.GetRoomParticipants(name) }`. Return that list.
- In `POST /rooms`, replace the `RoomListUpdated` broadcast payload with the same enriched shape (use `PresenceHub.BuildRoomListPayload` over the room rows).
