# Plan 01-01: C# Server Foundation

## Objective

Build the C# ASP.NET Core 8 server that handles user authentication (JWT) and issues LiveKit room access tokens. The server is NOT in the audio path — LiveKit handles all WebRTC media routing. The server's role: authenticate users, generate LiveKit tokens, and track Pulse room presence via SignalR.

## Context

- Framework: ASP.NET Core 8 Web API
- Auth: JWT (HS256 access token 30 min, refresh token 7 days)
- DB: EF Core + SQLite (dev)
- LiveKit SDK: `Livekit.Server.Sdk.Dotnet` NuGet package
- SignalR: presence hub only (not WebRTC signaling — LiveKit owns that)

### Task 3: TokenService (JWT + refresh tokens)

<action>
Create `Pulse.Server/Services/ITokenService.cs`:
```csharp
public interface ITokenService
{
    string GenerateAccessToken(User user);
    Task<string> GenerateRefreshTokenAsync(Guid userId, CancellationToken ct = default);
    Task<User?> ValidateRefreshTokenAsync(string token, CancellationToken ct = default);
}
```

Create `Pulse.Server/Services/TokenService.cs`:
- `GenerateAccessToken`: HS256 JWT with claims `sub` (userId GUID), `displayName`, `iat`, `exp` (30 min)
- `GenerateRefreshTokenAsync`: 32-byte `RandomNumberGenerator` → base64 string, store in DB, return token string
- `ValidateRefreshTokenAsync`: find non-revoked, non-expired token in DB; mark it revoked; return the associated `User` (rotation: caller issues a new refresh token after this)
</action>

<acceptance_criteria>
- `GenerateAccessToken` produces a valid HS256 JWT signed with the key from `IConfiguration`
- `GenerateRefreshTokenAsync` stores a new `RefreshToken` row and returns the token string
- `ValidateRefreshTokenAsync` marks the token revoked and returns `null` for expired or revoked tokens
</acceptance_criteria>

### Task 4: AuthController

<action>
Create `Pulse.Server/Controllers/AuthController.cs` with `[ApiController]`, `[Route("auth")]`, `[AllowAnonymous]`:

- `POST /auth/register` — body: `{ displayName, password }`. Hash password with BCrypt, create User, call `GenerateRefreshTokenAsync`, return `{ accessToken, refreshToken, userId, displayName }`.
- `POST /auth/login` — body: `{ displayName, password }`. Find user by displayName, verify BCrypt hash, generate tokens, return same shape.
- `POST /auth/refresh` — body: `{ refreshToken }`. Call `ValidateRefreshTokenAsync`; if valid, issue new access token and new refresh token (rotation); return `{ accessToken, refreshToken }`. Return 401 if invalid.

All endpoints return `400` on validation errors (missing fields), `401` on auth failure.
</action>

<acceptance_criteria>
- `POST /auth/register` creates a user and returns tokens
- `POST /auth/login` with correct credentials returns tokens; wrong password returns 401
- `POST /auth/refresh` rotates the refresh token and returns new tokens; invalid token returns 401
- Passwords are stored as BCrypt hashes (never plaintext)
</acceptance_criteria>

### Task 7: PresenceHub (SignalR — Pulse room presence)

<action>
Create `Pulse.Server/Hubs/PresenceHub.cs`:

```csharp
[Authorize]
public class PresenceHub : Hub
{
    private static readonly ConcurrentDictionary<string, ConcurrentDictionary<string, string>>
        _rooms = new(); // roomName → { connectionId → displayName }

    public async Task JoinRoom(string roomName)
    {
        var displayName = Context.User?.FindFirst("displayName")?.Value ?? "Unknown";
        _rooms.GetOrAdd(roomName, _ => new()).TryAdd(Context.ConnectionId, displayName);
        await Groups.AddToGroupAsync(Context.ConnectionId, roomName);
        await Clients.Group(roomName).SendAsync("ParticipantJoined", Context.ConnectionId, displayName);
        var participants = _rooms.GetValueOrDefault(roomName, new())
            .Select(kv => new { connectionId = kv.Key, displayName = kv.Value });
        await Clients.Caller.SendAsync("RoomJoined", roomName, participants);
    }

    public async Task LeaveRoom(string roomName)
    {
        if (_rooms.TryGetValue(roomName, out var room))
            room.TryRemove(Context.ConnectionId, out _);
        await Groups.RemoveFromGroupAsync(Context.ConnectionId, roomName);
        await Clients.Group(roomName).SendAsync("ParticipantLeft", Context.ConnectionId);
    }

    public override async Task OnDisconnectedAsync(Exception? exception)
    {
        foreach (var (roomName, room) in _rooms)
        {
            if (room.TryRemove(Context.ConnectionId, out _))
                await Clients.Group(roomName).SendAsync("ParticipantLeft", Context.ConnectionId);
        }
        await base.OnDisconnectedAsync(exception);
    }
}
```
</action>

<acceptance_criteria>
- `JoinRoom` adds the connection to a SignalR group, sends `RoomJoined` to caller with current participant list
- `ParticipantJoined` is broadcast to the room when someone joins
- `LeaveRoom` removes the connection and broadcasts `ParticipantLeft`
- `OnDisconnectedAsync` cleans up all room memberships on drop
- Hub requires JWT auth
</acceptance_criteria>

### Task 8: Program.cs wiring

<action>
Configure `Program.cs`:

1. **DbContext**: `builder.Services.AddDbContext<AppDbContext>(o => o.UseSqlite("Data Source=pulse.db"))`
2. **Services**: `AddScoped<ITokenService, TokenService>()`, `AddScoped<ILiveKitService, LiveKitService>()`
3. **JWT Auth** with SignalR query-string token support:
   ```csharp
   builder.Services.AddAuthentication(JwtBearerDefaults.AuthenticationScheme)
     .AddJwtBearer(options => {
       options.TokenValidationParameters = new() {
         ValidateIssuer = true, ValidIssuer = config["Jwt:Issuer"],
         ValidateAudience = true, ValidAudience = config["Jwt:Audience"],
         ValidateIssuerSigningKey = true,
         IssuerSigningKey = new SymmetricSecurityKey(Encoding.UTF8.GetBytes(config["Jwt:Key"]!)),
         ValidateLifetime = true
       };
       options.Events = new JwtBearerEvents {
         OnMessageReceived = ctx => {
           var token = ctx.Request.Query["access_token"];
           var path = ctx.HttpContext.Request.Path;
           if (!string.IsNullOrEmpty(token) && path.StartsWithSegments("/hubs"))
             ctx.Token = token;
           return Task.CompletedTask;
         }
       };
     });
   ```
4. **SignalR**: `builder.Services.AddSignalR()`
5. **CORS**: `AllowAnyOrigin().AllowAnyHeader().AllowAnyMethod()` for dev (tighten in production)
6. **Controllers**: `builder.Services.AddControllers()`
7. **Startup DB**: `await db.Database.EnsureCreatedAsync()` using scoped `AppDbContext`
8. **Map**: `app.MapControllers()`, `app.MapHub<PresenceHub>("/hubs/presence")`
</action>

- `pulse.db` created on first run
